nod2svg
=======

A quick & simple utility to generate an SVG image from a Nodal_ document.

.. sourcecode:: bash

    $ nod2svg Nebula2.nod Nebula2.svg

You can install a development version from Github_.

.. sourcecode:: bash

    $ pip install nod2svg

.. _Nodal: http://www.csse.monash.edu.au/~cema/nodal/
.. _Github: https://github.com/emcconville/nod2svg
